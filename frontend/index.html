🎵 Connect Spotify

To use Spotify, you need a free Spotify Developer app. Takes 3 minutes:

  1. Go to dashboard.spotify.com → Log in → Create App
  2. App name: Bloomdoro — check Web API
  3. In Settings → Redirect URIs, add your app URL + /api/spotify/callback
  4. Copy your Client ID and Client Secret
  5. On Render dashboard → your service → Environment → add:
    SPOTIFY_CLIENT_ID = your_client_id SPOTIFY_CLIENT_SECRET = your_client_secret SPOTIFY_REDIRECT_URI = https://your-app.onrender.com/api/spotify/callback
  6. Redeploy on Render → come back and click Connect Spotify
Open Spotify Dashboard ↗
